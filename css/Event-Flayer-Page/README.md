# Event Flyer Page

A responsive event flyer webpage built using semantic HTML and CSS.

## Features
- Semantic HTML structure (header, main, section, footer)
- Image and event title in header
- Multiple feature sections with headings
- Responsive layout using viewport units (vw, vh)
- CSS `calc()` for dynamic min-height
- Centered layout with proper spacing
- Styled horizontal rule divider

## What I Practiced
- Semantic elements
- Viewport units (vw, vh)
- CSS `calc()` function
- Layout alignment using inline-block / flexbox
- Spacing and visual structure

## Technologies Used
- HTML5
- CSS3
