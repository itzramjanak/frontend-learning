# Blog Post Card

A modern blog post card component built using HTML and CSS.  
This project focuses on layout design, hover effects, and clean UI styling.

## Features

- Responsive card layout
- Centered using Flexbox
- Image with border accent
- Smooth hover animation
- Styled call-to-action button
- Box shadow for depth effect

## Technologies Used

- HTML5
- CSS3 (Flexbox, transitions, box-shadow)

## What I Practiced

- Structuring content using semantic HTML
- Centering elements with Flexbox
- Using transition and transform properties
- Styling buttons and hover states
- Creating modern card UI components
